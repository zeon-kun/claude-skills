# Design Component

Design a React component, section, or UI element from scratch.

Launches `frontend-component-designer` which:
- Asks 6 targeted questions about what you're building
- Chooses the right layout pattern (bento, grid, hero, dashboard, etc.)
- Applies your brand tokens (or shadcn defaults)
- Produces complete, accessible, dark-mode-ready JSX
- Layers in Framer Motion or GSAP animation if appropriate

## Usage
`/design-component <component name or description>`
